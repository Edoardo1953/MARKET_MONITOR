Privacy and Access Guide

Instructions for secure use of the FOREX Dashboard

1. Entering the Privacy Area

To ensure maximum security, we have separated aesthetic settings from protection settings.

  • On the Homepage, locate the Key icon.
  • Clicking the icon will open the dedicated secure sharing panel.

2. Creating an Access Link

Step A: Set Password: Enter a password of your choice. This acts as a cryptographic key and is not saved in our systems.
Step B: Choose Duration: Select how long you want the link to be active (e.g., 1 month). After this period, the link will automatically expire.
Step C: Generate Link: Click "Generate Link". A special web address (Token) will be created containing the encrypted expiry information.

3. How the Link Works for the User

When the recipient opens the link:

  1. The system detects the token and blocks access with a security screen.
  2. The password you initially set is requested.
  3. The user's browser attempts to decrypt the content. If the password is correct and the time hasn't expired, the dashboard is displayed.
Important: Since protection occurs entirely in the browser (client-side), if you forget the password for the generated link, it cannot be recovered. You will simply need to generate a new link.
