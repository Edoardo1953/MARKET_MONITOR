Advanced User Manual

FOREXcross Dashboard - Professional Version 2.5

INTRODUCTION

Welcome to FOREXcross, the definitive financial analysis tool for monitoring exchange rates. Version 2.5 introduces advanced features for user experience personalization, data security, and deep historical analysis.

1. INTERFACE AND LAYOUT

The Dashboard is designed to adapt dynamically to every device:

Desktop & Tablet

Expanded view with a top control menu, interactive charts, and monthly tables on a single screen for maximum productivity.

Mobile Mode (CELL)

Optimized layout for smartphones. All controls are accessible via a compact interface that minimizes scrolling.

💡 Developer Function: If you are using the app from a PC, you can activate the Mobile Preview (CELL) via the shortcut key in the lower-right corner. This allows you to instantly see how the dashboard appears on a smartphone.

2. CROSS SEARCH AND ANALYSIS

A. Personalized Calculation (Cross Search)

Use the central module to calculate the exchange rate between any two currencies. Click on the EUR or USD fields to bring up the list of available currencies. Once selected, the chart and historical data will update instantly.

B. Interactive Charts

The main chart allows you to filter the temporal trend:

  • 1M to 1Y: For short and medium-term analysis.
  • 5Y and History: For a macro view of currency trends from 2010 to the present.

3. DAILY HISTORICAL DATABASE

By accessing the "DATABASE" section in the top menu, you can consult every single daily quote from 2010 to today:

  • Period Filter: Select a date range (Month/Year) to narrow your search.
  • Month End (Monthly Closures): Activate this filter to view only the yellow rows (the last working day of each month), essential for accounting reporting.
  • Excel Export: Download the entire filtered database directly in .xlsx format, ready for professional use.

4. SECURITY AND SHARING

Through the Key icon (Privacy), you can generate protected access for collaborators or clients:

  1. Set an Access Password of your choice.
  2. Select a Duration (from 1 month to 1 year).
  3. Generate the Access Link: send it to anyone you wish. The recipient will need to enter your password to view the data. Once the set time expires, the link will automatically stop working.
🔒 Client-Side Security: All passwords are managed in the browser. If you forget the password for a generated link, you cannot recover it: you will simply need to generate a new one.

5. TOOLS AND PERSONALIZATION

In the Gear icon (Settings), you will find:

  • Themes: Choose between Light, Dark, or "Total Black" themes to reduce eye strain or save battery on OLED screens.
  • Clear Cache: Useful for forcing data redownload if you notice discrepancies or want to clear the device's local memory.
  • Manuals: Quick access to guides (like this one) that you can download or print.

6. DOWNLOAD AND PRINT PDF

Every manual has a **"Download / Print PDF"** button. By clicking it:

  1. The system's print window will instantly open.
  2. You can choose your physical printer for a paper copy.
  3. Or select "Save as PDF" to download the manual digitally.
