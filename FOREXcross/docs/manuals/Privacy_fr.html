Guide de Confidentialité et d'Accès

Instructions pour une utilisation sécurisée du FOREX Dashboard

1. Accéder à la zone de confidentialité

Pour assurer une sécurité maximale, nous avons séparé les paramètres esthétiques des paramètres de protection.

  • Sur la Page d'accueil, repérez l'icône en forme de Clé.
  • En cliquant sulla icône, vous ouvrirez le panneau de partage sécurisé dédié.

2. Création d'un lien d'accès

Étape A : Définition du mot de passe : Saisissez un mot de passe de votre choix. Il s'agit d'une clé cryptographique qui n'est pas enregistrée dans nos systèmes.
Étape B : Choix de la durée : Sélectionnez la durée pendant laquelle vous souhaitez che le lien soit actif (par exemple, 1 mois). Passé ce délai, le lien expirera automatiquement.
Étape C : Génération du lien : Cliquez sur "Générer le lien". Une adresse web spéciale (Token) sera créée, contenant les informations d'expiration cryptées.

3. Comment fonctionne le lien pour l'utilisateur

Lorsque le destinataire ouvre le lien :

  1. Le système détecte le jeton et bloque l'accès par un écran de sécurité.
  2. Le mot de passe que vous avez initialement défini est demandé.
  3. Le navigateur de l'utilisateur tente de décrypter le contenu. Si le mot de passe est correct et che le délai n'est pas dépassé, le tableau de bord s'affiche.
Important : Étant donné que la protection s'effectue entièrement sur le navigateur (côté client), si vous oubliez le mot de passe du lien généré, il ne pourra pas essere récupéré. Vous devrez simplement générer un nouveau lien.
