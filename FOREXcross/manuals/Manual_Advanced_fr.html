FOREXcross Dashboard - Version Professionnelle 2.5

INTRODUCTION

Bienvenue dans FOREXcross, l'outil d'analyse financière définitif pour le suivi des taux de change. La version 2.5 introduit des fonctionnalités avancées pour la personnalisation de l'expérience utilisateur, la sécurité des données et l'analyse historique approfondie.

1. INTERFACE ET MISE EN PAGE

Le tableau de bord est conçu pour s'adapter dynamiquement à chaque appareil :

Ordinateur & Tablette

Vue étendue avec menu de contrôle supérieur, graphiques interactifs et tableaux mensuels sur un seul écran pour une productivité maximale.

Mode Mobile (CELL)

Mise en page optimisée pour smartphone. Tous les contrôles sont accessibles via une interface compacte qui minimise le défilement.

💡 Fonction Développeur: Si vous utilisez l'application depuis un PC, vous pouvez activer l'Aperçu Mobile (CELL) via la touche raccourci dans le coin inférieur droit. Cela vous permet de voir instantanément comment le tableau de bord apparaît sur un smartphone.

2. RECHERCHE CROSS ET ANALYSE

A. Calcul Personnalisé (Cross Search)

Utilisez le module central pour calculer le taux de change entre deux devises quelconques. Cliquez sur les champs EUR ou USD pour afficher la liste delle devises disponibles. Une fois sélectionnées, le graphique et les données historiques se mettront à jour instantanément.

B. Graphiques Interactifs

Le graphique principal permet de filtrer la tendance temporelle :

  • 1M à 1Y: Pour l'analyse à court et moyen terme.
  • 5Y et Historique: Pour une vue macro des tendances monétaires de 2010 à aujourd'hui.

3. BASE DE DONNÉES HISTORIQUE QUOTIDIENNE

En accédant à la section "DATABASE" dans le menu supérieur, vous pouvez consulter chaque cotation quotidienne de 2010 à aujourd'hui :

  • Filtre Période: Sélectionnez une plage de dates (Mois/Année) pour restreindre votre recherche.
  • Fin de Mois (Clôtures Mensuelles): Activez ce filtre pour afficher uniquement les lignes jaunes (le dernier jour ouvrable de chaque mois), essentiel pour les rapports comptables.
  • Exportation Excel: Téléchargez l'intégralité de la base de données filtrée directement au format .xlsx prêt per un usage professionnel.

4. SÉCURITÉ ET PARTAGE

Via l'icône de la Clé (Privacy), vous pouvez générer des accès protégés pour des collaborateurs ou des clients :

  1. Définissez un Mot de passe de votre choix.
  2. Sélectionnez une Durée (de 1 mois à 1 an).
  3. Générez le Lien d'Accès : envoyez-le à qui vous souhaitez. Le destinataire devra saisir votre mot de passe pour voir les données. Une fois le délai expiré, le lien cessera automatiquement de fonctionner.
🔒 Sécurité Côté Client: Tous les mots de passe sont gérés dans le navigateur. Si vous oubliez le mot de passe d'un lien généré, vous ne pourrez pas le récupérer : vous devrez simplement en générer un nouveau.

5. OUTILS ET PERSONNALISATION

Dans l'icône de l'Engrenage (Settings), vous trouverez :

  • Thèmes: Choisissez entre les thèmes Clair, Sombre o "Noir Total" per réduire la fatigue visuelle o économiser la batterie sur les écrans OLED.
  • Effacer le Cache: Utile pour forcer le retéléchargement des données si vous remarquez des écarts ou souhaitez vider la mémoire locale de l'appareil.
  • Manuels: Accès rapide aux guides (comme celui-ci) que vous pouvez télécharger ou imprimer.

6. TÉLÉCHARGEMENT ET IMPRESSION PDF

Chaque manuel dispose del touche **"Télécharger / Imprimer PDF"**. En cliquant dessus :

  1. La fenêtre d'impression du système s'ouvre instantanément.
  2. Vous pouvez choisir votre imprimante physique pour une copie papier.
  3. Ou sélectionnez "Enregistrer au format PDF" pour télécharger le manuel numériquement.
